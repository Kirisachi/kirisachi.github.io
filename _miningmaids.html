MiningMaid's Crypto Miners

Being apart of crypto mining at the time we needed some more dedicated miners at the time

The main problem was that there was not enough information out at the time and we needed to experiement if each motherboard would be able to handle and let us use more than 3 or more graphics cards in one unit.

our first motherboard blew and burned the 24 pin atx female connector on the motherboard and also burned the psu side of the 6+2 PCIE female connector (Sorry I only had a photo of the PSU(Powersupply))

We found the solution to the Mobo(Motherboard) exploding problem for the Powersupply it was caused by overvolting and error

The second motherboard was a semi success now the problem with this computer was that, not all the GPUs were getting detected which caused us problem since we needed all the gpus to work

in order to have for all the gpu's to work we first tried to install the graphics drivers. we then would restart the whole system consistenly (total around 30 times)

And somehow after all the restarting the system started to detect the GPUs which solved our problem to get more than 3 gpus on one system
