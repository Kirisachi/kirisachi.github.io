KN Network

KN Network is a network for server hosting based in Australia.


Having to go through high ping servers to have a fun game of minecraft or iW4x really irritated me.
My first thought was "Why not make my own server network"
Using Windows Server as my os (GUI), my rack mount servers with 128gbs of ram each server and my domain, I brought off namecheap.com
I Was ready to setup my servers!
For the Website I used apache and for my game servers I had server one host the games with their dedicated server files. Server two was being used as a private media server but can still be accessed from the public

jellyfin.knnetwork.xyz


Anyways I had to reverse proxy the media server so i could host it on the site
and that was it! my servers were up.
